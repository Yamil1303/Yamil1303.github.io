Typography

Heading Font: [Georgia, serif]

[The reason why I chose this font is that this font convey more clarity to the readers so that they can be persuaded to continue to reading. Since this font is similar with Times New Roman, has a common usage with an good style.]

Paragraph Font: [Tahoma, Geneva, sans-serif]

[I chose this font because it is clear for the reader. This font shows a cleaner and more elegant text so that whoever want to read it, can be encouraged to keep reading. The size of the letters are precise so that it will not be challenging for others to read.]
